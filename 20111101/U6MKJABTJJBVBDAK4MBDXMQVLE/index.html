煮題　玫瑰園

有料到
乾玫瑰花　保濕美顏

清香的玫瑰，能治療憂鬱情緒、女士經痛及腸胃病，更具保濕美顏作用。

粟膠　令食材煙韌

由粟米提煉而成，質地像果糖，帶微甜，可提升食材黏性，常用於製作糖果。

忌廉芝士　味道香濃

用牛奶做成，以細菌發酵而成。味道香濃，質地光滑柔軟，是製造芝士餅的完美材料。

淡忌廉　做慕絲蛋糕

打至出現紋理即可，一般可發至兩倍大體積，其味道香濃，適用於製作慕絲蛋糕及芝士餅。

泡打粉　蛋糕發粉

又名蛋糕發粉，由蘇打粉配合其他材料製成的膨脹劑，與水溶合後經烘焙，能使蛋糕及西餅鬆軟。

梳打粉　中和酸性

與泡打粉用法不同，因它屬鹼性，配合酸性食物如朱古力，能使西點膨脹外，亦能中和朱古力酸性，使朱古力更光亮。

玫瑰美人杯子蛋糕

這個玫瑰杯子蛋糕，用忌廉芝士唧出一朵靚玫瑰，味道不太甜，真是內外兼備！

材料

低筋麪粉（過篩）/無鹽牛油（放軟）/忌廉芝士（放軟）各100克、泡打粉（過篩）1茶匙、砂糖/糖粉各60克、鹽少許、牛奶50毫升、雞蛋（拂勻）1隻、水適量、食用色素（紅）3滴

做法

1. 將70克無鹽牛油用電動打蛋器打至淡色，且呈乳霜狀，加砂糖、鹽打勻，下蛋液再打勻。

2. 分三次加入低筋麪粉和泡打粉，每次用膠刮拌勻後加入1/3份牛奶，拌至滑身，倒進紙杯至一半滿，放入預熱至180℃焗爐，焗20分鐘，待凉。

3. 忌廉芝士、半份糖粉、餘下無鹽牛油、食用色素以打蛋器打至光滑乳霜狀，成裝飾忌廉。

4. 餘下糖粉加適量水，塗在蛋糕面，放凉。

5. 將裝飾忌廉放入唧袋，在蛋糕上唧出玫瑰花形，最後撒上餘下糖粉裝飾即可。

貼士：

麪糰攪拌過度會影響膨脹效果，所以攪拌動作一定要快，拌勻就可以了。

情迷玫瑰曲奇

曲奇的牛油香在口腔內散發，隱約間嚐到玫瑰香，味道非常搭調，相信大家都會喜歡!

材料

無鹽牛油（放軟）100克、糖粉50克、鹽/梳打粉各1/4茶匙、低筋麪粉180克、乾玫瑰花10粒、玫瑰醬1茶匙

做法

1. 撕碎玫瑰花瓣，取走花芯，花瓣備用。

2. 牛油、糖粉、鹽以膠刮拌勻，用電動打蛋器快速打勻，加入玫瑰醬再打。

3. 分數次篩入低筋麪粉及梳打粉，每次輕輕攪拌成為麪糰。

4. 下花瓣略搓壓，包上保鮮紙，用酥棍搓成約5毫米厚，置雪櫃冷藏約30分鐘至麪糰硬身，取出麪糰後撕開保鮮紙。

5. 玫瑰形模具沾上低筋麪粉，在麪糰上印出圖案，放上鋪有焗爐紙的焗盤上，放入預熱160℃焗爐，焗10分鐘，再調細至150℃，焗15分鐘，取出放凉即可。

貼士：

爐溫切勿太高，否則難以保持玫瑰花的色澤及香味。

玫瑰忌廉蛋糕

這個三層玫瑰忌廉蛋糕，每層夾着清香的玫瑰忌廉，清新的感覺，讓人想起那些年，我們一起追的女孩!

材料

砂糖35克、雞蛋2隻、低筋麪粉40克、無鹽牛油（已溶）各20克

玫瑰忌廉：乾玫瑰花20克、食用色素（紅色）少許、砂糖50克、甜忌廉150克、水250毫升、粟膠4湯匙

做法

1. 先製作玫瑰忌廉，煲滾水，放玫瑰花，以慢火煮5分鐘，熄火後，冚蓋焗5分鐘至玫瑰花出味，隔去玫瑰，倒出花茶。

2. 鍋中倒入花茶，開大火，加入砂糖、粟膠拌勻煮沸，轉慢火煮15-20分鐘至起泡及稠身，放凉成玫瑰糖漿。

3. 甜忌廉用電動打蛋器打至企身，加入玫瑰糖漿、食用色素，輕輕拌勻，用保鮮紙包好，放入雪櫃冷藏30分鐘成玫瑰忌廉。

4. 雞蛋拂勻，下砂糖，用電動打蛋器打至稠身，分兩次篩入低筋麪粉略拌，加入牛油拌成麪糊，倒入鋪上牛油紙的焗盤中，放入預熱180℃焗爐，焗15分鐘，取出成蛋糕。

5. 用心形模印出3個心形蛋糕，於每一片蛋糕邊唧上玫瑰忌廉，叠成3層即可。

貼士：

如愛玫瑰忌廉的味道，可塗更多忌廉於蛋糕上，那就更可口了。

甜蜜蜜玫瑰芝士餅

香濃芝士配上誘人的蜜糖和清新的玫瑰，令這個蜜糖玫瑰芝士餅濃情滿溢！

材料

消化餅碎160克、無鹽牛油（已溶）50克、朱古力（已溶）/蜜糖各80克、乾玫瑰花40克、牛奶200毫升、魚膠粉（已溶）5克、水15毫升、忌廉芝士/淡忌廉各100克、砂糖60克

做法

1. 消化餅碎、無鹽牛油拌勻，加入已溶朱古力拌勻，放入已包好牛油紙的蛋糕模底，壓平，放入雪櫃冷藏30分鐘成餅底。

2. 煲中下牛奶、玫瑰花，開中火煮5分鐘，熄火再焗5分鐘，用篩隔去玫瑰，倒出玫瑰奶，備用。

3. 忌廉芝士、砂糖用電動打蛋器打至滑身，加入已打成出現紋狀的淡忌廉，拌成芝士漿。

4. 魚膠粉、水混合，加入1湯匙芝士漿拌勻，然後倒回芝士漿中拌勻。

5. 玫瑰奶倒入芝士漿中拌勻，加蜜糖再拌，倒入蛋糕模中，冷藏一小時即可脫模。

貼士：

煲玫瑰奶時不要太大火，否則牛奶容易煲焦。

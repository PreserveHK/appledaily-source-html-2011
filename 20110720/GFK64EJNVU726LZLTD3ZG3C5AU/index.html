煮題　泰凉菜
有料到
椰糖　泰國產味淡

分泰國和印尼，印尼品種椰香味較濃，可在超市買到；泰國的則較清淡，可於泰國雜貨舖找到。

蘆菇　浸水去昆蟲

果實間易藏小昆蟲，甚至有蠍子，買回來後要浸水中小心清洗。

青瓜　減肥美白

有利水、清熱、止渴、潤腸通便、減肥之功效，此外，含維他命C，具有阻止黑色素沉澱而達致美白的效果。

烏頭魚　健脾益氣

肉甘甜，含蛋白質、脂肪，健脾益氣，可調理脾虛泄瀉，消化不良及貧血等症狀。

香茅　治皮膚病

新鮮或乾燥的均具怡人檸檬香，適宜入饌外，也可用來治皮膚病、祛風止癢、消腫止痛等功用。

海底椰　止咳養顏

原產非洲，具清燥熱、止咳養顏功效。其果肉味淡而清甜，靚的海底椰色澤較白淨。

辣汁蘆菇雞絲撈

天時暑熱，吃炸雞扒有點膩滯，不過加入青瓜和蘆菇撈勻拌吃，清新好味一啲都唔膩。

材料

蘆菇454克、雞扒2件、青瓜1條、花生碎2茶匙

調味料：鹽1/4茶匙、糖1茶匙、胡椒粉少許、生粉1湯匙、沙薑粉2茶匙

辣汁：糖/白醋各2湯匙、豆瓣醬1湯匙、蒜茸1茶匙、魚露1/2茶匙

做法

1. 雞扒下調味料拌勻，醃約20分鐘。

2. 青瓜切絲；蘆菇浸水中逐一摘下，去殼取肉，分成小瓣備用。

3. 燒滾油至180℃，以細火將雞扒炸4-5分鐘，轉大火炸約2分鐘至黃金熟透，撈起瀝油後切條。

4. 將糖、豆瓣醬、蒜茸、醋、魚露拌勻，加入青瓜絲、蘆菇、雞條撈勻上碟，最後撒上花生碎即成。

香茅杞子凍飲

怪師傅以香茅來做飲品，還加入了杞子、海底椰，清熱化痰止咳明目樣樣得，消暑一流！

材料

香茅5支、杞子/椰糖各20克、水2公升、海底椰1個、青檸4片

做法

1. 海底椰略浸水中，刮去外皮，切薄片。

2. 椰糖略拍碎；杞子先浸暖水至軟身備用。

3. 香茅略拍尾部，切段。

4. 煲滾水放入香茅、海底椰，冚上蓋以細火煮約20分鐘，取出香茅，加入椰糖煮溶，熄火，加入杞子。

5. 倒出杞子香茅水，放雪櫃雪凍後倒入杯中，放上香茅及青檸片裝飾即成。

檸蜜凍烏魚

泰國人喜歡吃烏頭魚，貪其啖啖肉又少骨，用來做凍魚，蘸上酸甜汁來吃就最適合。

材料

烏頭魚1條、青檸1個、香茅2支

蘸汁：蜜糖4湯匙、蒜茸/普寧豆醬各1湯匙、指天椒1隻、青檸汁2茶匙

做法

1. 青檸開半。

2. 香茅放上碟，放上烏頭魚，塞入青檸，以大火隔水蒸約15分鐘，取出待凉。

3. 指天椒切碎後，加入蒜茸、青檸汁、普寧豆醬、蜜糖拌勻。

4. 將烏魚放雪櫃雪凍，取出，享用時蘸上汁料。

貼士

魚腹腔是腥味根源，放青檸入內可有效辟除腥臭味。

泰式五青肉鬆

五青汁大家飲得多，用青椒、西芹、青蘋果、苦瓜及皇帝芒這五青煮餸，配上煮好的肉鬆，清甜爽脆。

材料

蒜茸1茶匙、肉碎200克、皇帝芒1個、青蘋果1/2個、苦瓜（切條）100克、青椒（切條）/洋葱（切絲）各50克、西芹（切片）80克、蝦片10片

醋汁：白醋200毫升、糖6湯匙、魚露1茶匙

調味料：鹽1/4茶匙、糖/老抽各1茶匙、生粉2茶匙、水1湯匙

做法

1. 皇帝芒切開去皮起肉，切片；青蘋果去芯後切片。

2. 白醋加入糖、魚露拌勻，加青蘋果、芒果、西芹、苦瓜、青椒、洋葱拌勻浸約15分鐘後夾起，瀝水。

3. 燒滾油至140℃，分幾次放入蝦片，炸至發起即取出瀝油。

4. 肉碎加入調味料拌勻略醃。

5. 開大火燒熱鑊下油，下蒜茸、肉碎炒散至熟後，盛起，倒入（2）拌勻。

6. 蝦片圍碟，放上肉碎蔬菜料即成。

貼士

青蘋果切好後下醋汁，可防止氧化，仲可變雪白。

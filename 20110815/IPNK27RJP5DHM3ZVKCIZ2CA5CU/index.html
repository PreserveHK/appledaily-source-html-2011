煮題　免油炸香口菜式

有料到
豬手　骨細肉多

即豬前蹄，骨架和關節位較後蹄小，肉較厚，含豐富膠原蛋白，可增加皮膚彈性。

百里香　烹調先放

氣味芳香卻不太濃烈，需較長時間釋放香味，故用前宜撕碎，並於開始烹調時放入。

核桃仁　清除膽固醇

其脂肪有清除膽固醇功效，但要留意每天吃大約5粒便夠，以免攝入過高熱量。

新薯　口感較實

薯仔嬰兒，皮色黃中帶淡綠，水份多而澱粉質較少，口感較焗薯實身。

迷迭香　泡茶減頭痛

多用作烹調肉類，尤以味道較濃的羊肉最夾，可取迷迭香泡茶，有增強腦部功能和改善頭痛功效。

粟米仔　排列緊密

又叫玉米笋或珍珠笋，在粟米仍在幼嫩時便摘下，選擇時要完整呈圓椎形、笋粒飽滿且排列緊密細緻，淡黃色為佳。

熱風吹得脆

這部免油健炸鍋用上專利的快速空氣技術（RapidAirTechnology），以高速循環流動的空氣包圍食物，加上食物籃上的烤焗裝置極速加熱空氣，並在炸鍋底盤的特製坑紋配合下，於短時間內全面均勻地烹調食物，輕易煮出金黃香脆的美食，效果媲美傳統油炸！

炸魚薯條

傳統油炸薯條，食物完全浸在熱油中，吸收大量的油份，健炸鍋利用快速空氣技術烹調，可以減少一半油份，但仍保留到薯條的香脆效果。

材料

青衣白魚肉200克、玉米片30克、雞蛋（拂勻）1隻、新薯300克、植物油1湯匙、檸檬汁1/2湯匙、鹽/黑胡椒/沙律菜各少許

做法

1. 魚肉切4份，加入檸檬汁、鹽和黑胡椒醃5分鐘。

2. 玉米片放入保鮮袋中，用麪棍打碎。魚片先浸蛋液，再平均沾上玉米片。

3. 薯仔削皮、切條，用廚紙吸乾水，掃上植物油。

4. 將薯條和魚片分別放入分隔開的炸籃，放入預熱至200℃炸鍋中，計時器設定為12分鐘，取出配沙律菜同吃即可。

貼士

薯條塗油前要用廚紙抹乾，抹去澱粉質和多餘水份，炸出來會更香脆。

無骨雞翼蔬菜卷

一隻雞翼原來有6克脂肪，以健炸鍋來焗，可減少一半脂肪，而且雞皮香脆不油膩。

材料

雞中翼6件、泰國蘆笋/粟米仔各2條、甘笋（切條）25克、百里香（切段）2棵、黑胡椒碎/植物油各少許、鹽1/2茶匙

做法

1. 先用剪刀輔助起出雞翼內的骨頭。

2. 粟米仔切開4邊；蘆笋切2段。

3. 將蘆笋、甘笋條、粟米仔、百里香釀入雞翼中，並於雞翼表面灑上鹽、黑胡椒碎。

4. 蔬菜表面薄薄掃上植物油，放入已預熱至180℃的健炸鍋中，烤8至10分鐘即成。

貼士

蔬菜很易烤乾身，塗一層油便可保持光潤和鎖水。

香草烤豬手

一隻豬手未煮前有42克脂肪，用健炸鍋將豬手油份逼出後，脂肪足足減少四成！

材料

豬手（約600克）1隻、迷迭香/百里香4棵、鹽2茶匙、黑胡椒碎/沙律菜各少許、橄欖油1湯匙

做法

1. 用刀在豬手劃數刀。

2. 迷迭香和百里香撕碎，混入鹽、黑胡椒碎、橄欖油拌成醃料。

3. 醃料塗於豬手面，香草塞入割開位置，風乾30分鐘。

4. 豬手放入預熱至180℃健炸鍋內，設定時間為30分鐘。

5. 反轉豬手，再設定時間為30分鐘，取出配沙律菜即可。

貼士

豬手醃過後風乾，可令表皮更乾身，烘烤後更脆。

乾果朱古力布朗尼

用免油健炸鍋來做的布朗尼，外脆內軟，只要焗約20分鐘，就可輕鬆完成！

材料

純黑朱古力（切塊）/牛油（切塊）/糖各75克、蛋1隻、自發粉40克、核桃仁（切碎）25克、鹽少許

做法

1. 朱古力和核桃仁分別切碎。朱古力和牛油放入小煲中，以細火煮溶，其間不時攪拌，放凉備用。

2. 蛋拂勻，加糖和鹽拌勻，下朱古力混合物、自發粉和核桃碎，拌勻成麪糊。

3. 蛋糕盤鋪上焗盤紙，倒入麪糊，抹平。

4. 將蛋糕盤放入預熱至180℃的健炸鍋中，計時器設定為18分鐘，冷卻後切塊即可上碟。

貼士

做布朗尼時，可用6吋直徑的蛋糕盤，效果會更加好！

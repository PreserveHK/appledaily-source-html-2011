天然代糖羅漢果　絲絲入饌甜入心

常見的乾羅漢果都用來煲湯，但對着新鮮的羅漢果，大家便無從入手。原來鮮品水份和纖維較多，煮汁或取絲絲果肉便能煮出百變菜式。更重要的是，羅漢果是天然代糖，能降血糖，做甜點可減糖量，大家千萬不要錯過與這位「神果」的邂逅!

攝影：劉永發

羅漢果降血糖

羅漢果是桂林特產，因其營養價值豐富，功效多元化，所以被披上「中華神果」之美譽。它不但能潤肺化痰、滋陰補腎、有益腸臟，而且抗氧化功能超卓，能全面提升免疫力。它含有的維他命C，更是鮮橙的7倍。不過，很多肥胖或糖尿病患者對羅漢果的糖份存有不少誤解，其實，羅漢果的甜味源自其特有的羅漢果甜苷，多項研究證實有降血糖之效，絕對是糖尿病患者的好朋友。

新鮮羅漢果

味道清甜，入口甘凉略帶酸。除可加薑煲水飲用外，亦能做菜或甜品。羅漢果與酸味的醬汁最為搭調，煮成羅漢果水或榨汁，配糖醋汁，效果非常好。做甜品方面，可煲羅漢果水入饌，也可取其絲絲果肉成果茸，加入甜品中，份量不用多，因鮮羅漢果香味甜度已很突出。

乾羅漢果

味道濃甜，宜用來煲湯，配枇杷果、菜乾、無花果、南北杏、豬肺煲湯，有清熱潤肺的功效。其實，乾品與鮮品的營養價值相同，可是羅漢果製乾時需於高溫中烘焙48小時以上，過程中降解其健康物質。

點揀好？

顧客選購羅漢果時經常會在桌上彈一彈，選有聲的，但其實大錯特錯。無論新鮮或乾貨，選購時應搖一搖，沒有聲響的為上品。而新鮮的羅漢果，要選毛茸茸、金黃色，而且外殼硬實的，代表羅漢果已到最佳成熟期，味道最為清甜。

羅漢果肴肉拼燒汁黑豚

用羅漢果絲絲果肉做成的燒汁，其清甜味減少了黑豚的油膩感，佐以羅漢果肴肉，其甘凉口感更給人一種冰鎮的感覺，與黑豚肉非常搭調，口感味道一百分！

材料

羅漢果肴肉：新鮮羅漢果1個、熱水150毫升、魚膠片（浸軟）1片、大菜絲少許、黑豚肉（汆水切碎）200克

醬汁：西芹水（西芹煲水）/洋葱（切碎）/花椒/八角/芫荽（切碎）/紅蘿蔔（切片）/OK汁/茄汁/鹽/雞粉/蒜茸/紅椒（切碎）/青椒（切碎）各少許、水900毫升

醃料：雞粉/鹽/糖各少許、雞蛋（拂打）1隻、生粉適量

其他：黑豚肉3塊

做法

1. 羅漢果棄殼後放入攪拌機中，加入熱水攪成汁，隔去渣滓，取汁備用。

2. 將130毫升的羅漢果汁加入魚膠片拌溶，下黑豚肉、大菜絲拌溶，放入雪櫃冷藏1-2小時至凝固，取出切件即成。

3. 鍋中下西芹水、洋葱碎、花椒、八角、芫荽、紅蘿蔔、水，以中火煲半小時成600毫升份量，下OK汁、茄汁、鹽、雞粉再煮熱，備用。

4. 燒熱鑊下油，爆香蒜茸、洋葱碎、紅椒、青椒，加入餘下的羅漢果汁和（3）煮熱成醬汁，盛起備用。

5. 黑豚肉落雞粉、鹽、糖、生粉、雞蛋醃15分鐘。

6. 燒滾油，黑豚肉拍上適量生粉，放入油中半煎炸至兩面金黃色，盛起後蘸上醬汁即成。

羅漢果咕嚕斑片

羅漢果的清甜與酸酸甜甜的糖醋汁最為搭調，配搭斑片更能帶出魚的鮮味！

材料

羅漢果（切碎）1個、水600毫升、片糖150克、OK汁110克、白醋300毫升、茄汁220克、斑塊300克、鹽少許、紅椒（切件）/青椒（切件）/菠蘿（切粒）各適量、紅色色素1滴、酸梅子2粒、生粉芡適量

醃料：雞粉/鹽/糖/各少許、雞蛋（拂打）1隻、生粉適量

做法

1. 鍋下羅漢果和水，煮約半小時至300毫升，先盛起1湯匙羅漢果水，留用；加入片糖、鹽、酸梅子、白醋、茄汁、OK汁煮滾，加入紅色色素，隔走材料，成糖醋汁備用。

2. 斑塊下雞粉、鹽、糖、雞蛋，醃15分鐘，沾生粉備用。

3. 燒滾油約150℃，下斑塊拌炸至硬身，盛起上碟。

4. 原鑊倒去油，下紅椒、青椒、菠蘿略炒，加入1湯匙羅漢果糖醋水及1湯匙羅漢果水，加入適量生粉芡略煮，淋在斑塊上即成。

貼士

煮糖醋汁時切勿煮太長時間，否則容易煮焦。

羅漢果西米布甸

冬天最佳甜品莫過於暖笠笠的西米焗布甸，若略嫌有點悶蛋，試試周師傅新創的羅漢果西米布甸！羅漢果甘甘甜甜，令布甸即刻升晒呢！

材料

糖220克、西米/椰汁各150克、奶粉/吉士粉各75克、羅漢果半個、生粉/栗子茸/牛油（已溶）各40克、水1,200毫升

做法

1. 吉士粉加入奶粉、生粉，分數次加入椰汁拌勻，備用。

2. 羅漢果加水煮滾後，隔去渣，下西米，再煮10-15分鐘，下糖和牛油，再煮至滾起，熄火，一邊快速攪拌一邊倒入（1）中，拌成布甸漿。

3. 先將1/3布甸放入焗杯底，栗子茸壓扁放在布甸上，再倒入1/3布甸至九成滿，放入預熱至250℃焗爐焗8分鐘，取出即成。

貼士

椰汁要分數次加入，否則會容易起粒，難以做出細滑口感。

羅漢果奶皇千層糕

原來羅漢果味道與奶皇非常夾，如海綿般鬆軟的蒸糕散發着羅漢果獨特的甘甜！

做法

皮漿：麪粉/糖各200克、雞蛋6隻、奶粉/吉士粉35克、食粉9克、泡打粉21克、高筋麪粉3克、油70克、花奶1湯匙、羅漢果肉1茶匙

奶皇：牛奶250毫升、糖/牛油/椰汁各500克、奶粉/吉士粉/生粉各70克、雞蛋6隻

做法

1. 先做皮漿，麪粉加入雞蛋、糖用電動打蛋器快速打勻，加入奶粉、吉士粉、泡打粉、食粉、高筋麪粉打勻，再加入花奶、油，用慢速打勻，加入羅漢果肉，拌勻備用。

2. 將吉士粉、奶粉、1/2椰汁、生粉、雞蛋，拌至軟滑成漿，備用。

3. 鍋內下牛奶、糖、牛油、餘下椰汁煮滾，一邊加入（2）一邊快速攪拌至濃稠軟滑，熄火成奶皇，備用。

4. 抹油在鋪了焗爐紙的焗盤上，倒入皮漿抹平，放入鑊以大火隔水蒸5-6分鐘，取出後脫去焗爐紙，抹上一層薄薄的奶皇，再分成多件，變成一層層成千層糕即成。

入行30年，曾於多間星級酒店及會所任職，早年移民澳洲開設餐廳，擅長為西餐美食加入亞洲元素。

煮題　DHA芥花籽油營養煎炒炸
有料到
火龍果　預防便秘

果皮中色素為甜菜紅素，具有抗氧化、抗自由基等作用，其豐富水溶性纖維可預防便秘、降低患腸道疾病風險兼具瘦身作用。

三文魚　降膽固醇

營養豐富，含不飽和脂肪酸Omega-3，能有效地降低膽固醇，保護心臟。

DHA芥花籽油　有助腦部發展

從無污染深海抽取微藻DHA成份製成，攝取足夠的DHA，對兒童腦部發展都佔很重要角色。

牛肝菌　柔滑細緻

口感柔滑細緻，略帶堅果香味及濃郁肉香，可加入意大利粉、焗飯、湯或作菜餚配料。

乾玫瑰花　舒緩情緒

香氣濃郁，清而不濁，具有活血化瘀、舒緩情緒的作用。

安格斯牛肉　肉軟多汁

油花豐富，肉質柔軟且多汁，其蛋白質含量高，脂肪含量低於普通牛肉。

玫瑰花香煮三文魚扒

三文魚有淡淡的花香，煎時記得要先把鑊燒熱，否則會黏底，而且要留意煮的時間不要太長，避免肉質變粗糙。

材料

DHA芥花籽油50毫升、三文魚扒250克、蘆笋3條

醃料：鹽/胡椒粉/檸檬汁各少許

汁料：乾玫瑰花/香片茶葉各3克、乾菊花8克、水125毫升、糖1湯匙、生粉水適量

做法

1. 在三文魚扒上灑上鹽、胡椒粉、檸檬汁備用。

2. 煲滾水把香片茶葉、乾菊花、乾玫瑰花煮至滾，下糖熄火後焗約2分鐘，隔渣倒出花茶備用。

3. 燒熱鑊，倒入DHA芥花籽油，把三文魚扒煎至兩面金黃色盛起。

4. 將（2）倒入鑊，以細火煮熱，下三文魚扒回鑊，再放蘆笋。

5. 將三文魚扒略煮1分鐘後，取出，蘆笋再煮約3分鐘取出，餘下的汁液以生粉水埋芡。

6. 三文魚上碟，放上蘆笋，淋上汁料，可隨意撒上乾玫瑰花片裝飾即成。

煎牛肉火龍果卷配柚子汁

新鮮火龍果，加上鐵質豐富的牛肉，再配柚子汁，味道清新。

材料

DHA芥花籽油60毫升、火龍果（去皮切條）1個、安格斯牛肉片400克、菠菜葉150克、海鹽/胡椒粉/生粉各少許、韭菜花數條

柚子甜醬汁：乾葱（切碎）2粒、柚子蜜2茶匙、甜醋2湯匙、鹽少許

做法

1. 牛肉片撒上海鹽、胡椒粉，放上火龍果條，捲起，塗上生粉封口。

2. 燒滾水，下鹽、少許芥花籽油，放入菠菜葉烚軟盛起。

3. 鍋內下油，將菠菜葉回煲，下少許鹽輕炒菠菜葉，取出備用。

4. 煲中下乾葱碎、甜醋、柚子蜜，用慢火煮約30秒，下鹽調味，攪勻成醬汁，隔渣備用。

5. 開大火燒熱鑊倒入芥花籽油，煎牛肉卷至七成熟，取出上碟。

6. 原鑊下韭菜花略炒至軟身上碟，放上牛肉卷，以菠菜葉伴碟，淋上少許醬汁即成。

貼士：

做柚子汁時，可略加一點鹽吊一吊味，令味道層次更豐富。

越式春卷配士多啤梨汁

春卷好香脆，加上新鮮做的藍莓沙律醬，蘸一蘸，大大減低油膩感。

材料

DHA芥花籽油900毫升、越式春卷皮2片、香蕉1隻、核桃100克、牛油蛋糕1件（70克）、椰絲/士多啤梨各適量、檸檬汁少許

醬汁：士多啤梨3粒、藍莓80克、沙律醬60克、糖半茶匙、水3湯匙

做法

1. 先做醬汁，將士多啤梨去椗，切片放入煲，放藍莓、水、糖拌勻，以細火約10分鐘煮爛成茸，盛起放凉後，加入沙律醬攪勻。

2. 核桃略切碎；蛋糕切細粒；香蕉去皮分半，切條再切粒。

3. 將蛋糕、核桃、香蕉拌勻，下檸檬汁拌勻成餡料。

4. 春卷皮用暖水浸軟，取出抹乾多餘水份，放上餡料，包起成圓卷。

5. 開大火燒滾油至180℃，放入春卷，炸約1分鐘後，調細火炸約5分鐘至金黃色即可取出瀝油。

6. 將春卷上碟，旁邊放上士多啤梨裝飾，醬汁料上碟，最後撒上椰絲即成。

貼士：

香蕉去皮後加入適量檸檬汁，防止變黑。

牛肝菌雜菜燴意粉

牛肝菌跟意大利粉向來是最佳拍檔，而做燴意粉的秘訣是烚意粉時間宜比平時少2分鐘，燴完就不會過腍。

材料

意大利粉/紅洋葱絲各50克、牛肝菌100克、西蘭花/泰國蘆笋各30克、香草醬1湯匙、蒜片/鹽/胡椒粉各1茶匙、白酒10毫升、忌廉200克、黃椒絲/紅椒絲各25克、DHA芥花籽油適量

做法

1. 煲滾水，下芥花籽油和鹽，放入意大利粉烚約5分鐘，取出隔水備用。

2. 西蘭花和蘆笋用熱水烚2分鐘後，取出後浸冰水，備用。

3. 開大火燒熱鑊下芥花籽油，放入蒜片、紅洋葱絲、紅椒絲、黃椒絲，灒白酒、忌廉，意大利粉回鑊，下牛肝菌、鹽、胡椒粉拌勻上碟。

4. 放上西蘭花、蘆笋，最後加上香草醬，吃時拌勻即可。

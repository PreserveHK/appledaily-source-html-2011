煮題　梳乎三文魚

有料到
三文魚　多吃更聰明

有淡水魚的肥美和鹹水魚的鮮味，含豐富奧米加三，有健腦、抗氧化功效。

梳乎厘　鹹甜皆宜

法文為Souffle，是Blowup放大意思。傳統有甜的果醬、朱古力外，還有以三文魚來做鹹食梳乎厘。

番荽　預防心臟疾病

即洋芫荽，含豐富維他命C、胡蘿蔔素、可預防心臟疾病。

味噌　久煮失香

由米麴子、大豆發酵成，有助腸道健康、抗氧化功效。日本人多是熄火後才加，因煮得耐會失香味。

木魚花　做日式湯底

由鰹魚烘乾後加工製成，清香帶鮮味，可用來做凉拌、壽司。儲存時要放在陰凉處，避免受潮。

京葱　辟魚腥味

辛辣且香味較重，有殺菌、消毒作用，但燥熱者不宜食用。煮熟後帶甜味，更可辟去魚的腥味。

1.三文魚梳乎厘

甜食的梳乎厘大家食得多了，鹹的梳乎厘你又食過未呢？今次用三文魚來做一個梳乎厘，入口即溶又軟滑。

材料

三文魚（切小粒）30克、蛋白2隻、番荽碎/洋葱（切碎）/乾葱碎各1/3茶匙、酸忌廉/橄欖油/麪粉各少許

麪糊：蛋黃1隻、麪粉1/2茶匙、鮮奶80毫升　

醃料：白酒/生粉各1茶匙、薑汁1/3茶匙、水2茶匙、鹽/黑椒碎各少許

調味料：鹽1/4茶匙、胡椒粉少許

做法

1. 三文魚加醃料醃15分鐘，備用。

2. 開大火，燒熱鑊落油下洋葱碎、乾葱碎、三文魚、番荽碎炒香，盛起備用。

3. 準備麪糊，將麪粉、10毫升鮮奶拌勻，再加入10毫升鮮奶、蛋黃拌成鮮奶蛋液。

4. 開中火，將餘下60毫升的鮮奶稍為煮熱，加入鮮奶蛋液，煮至稠身成麪糊。

5. 將麪糊與三文魚混合成三文魚麪糊備用。

6. 蛋白加鹽、胡椒粉打拂至企身，分三次加入三文魚麪糊拌勻。

7. 杯中掃上少許橄欖油，沾上少許麪粉，將多餘麪粉倒出，加入三文魚麪糊，放在焗盤上。

8. 在焗盤上加少許清水，放入預熱至180℃焗爐，焗10分鐘。取出後放上酸忌廉和裝飾即可。

貼士：

三文魚和其他材料須先炒，可減少水份，以免影響梳乎厘的口感。

2.三文魚蒸水蛋

三色蒸水蛋食得多，三文魚蒸水蛋你會做嗎？水蛋又香又滑，三文魚令蛋更鮮甜，令人食到停唔到口。

材料

三文魚（切大粒）80克、雞蛋（拂勻）3隻、葱花1茶匙、水300毫升

醃料：豉油/米酒/生粉各1茶匙、薑汁1/2茶匙、水2茶匙、鹽/黑椒碎各少許

調味料：鹽1/4茶匙、胡椒粉少許

做法

1. 三文魚粒加醃料，醃15分鐘備用。

2. 開大火煲滾300毫升水，加三文魚煮至稍熟，熄火後分別盛起魚肉和150毫升的湯，備用。

3. 蛋液加入湯、調味料，過篩後先倒q入碟中，再加入一半三文魚肉。

4. 開大火煲滾水，放入三文魚蛋液，轉小火，蓋上蓋，在鑊蓋旁插入木筷子疏氣，隔水蒸10分鐘。

5.蛋蒸至稠身時，放上另一半三文魚肉，注入餘下蛋液，再蒸8分鐘，取出後灑上葱花即成。

貼士：

蒸水蛋時必須蓋上蓋，但一定要用筷子撐起鑊邊疏氣，那麼蛋才會滑。

3.鮮三文魚他他伴香芒

三文魚他他是法國的經典菜式，加上香芒，令菜式更清新。材料雖多，但做法簡單，是宴客之選。

材料

三文魚/芒果肉（切粒）各50克、洋葱（切粒）1茶匙、青瓜（切粒）25克、西芹（切粒）/番茄（切粒）/沙律菜各20克、番荽碎/黑醋/黑椒碎/橄欖油/鹽各少許

醃料：鹽/糖各1/3茶匙、檸檬汁/芥末醬各1/2茶匙、黑椒碎/白酒各1/4茶匙、橄欖油1茶匙、黑醋少許

裝飾：青瓜數片、小番茄（切半）1粒、檸檬1/4個、薄多士1片、黑醋少許

做法

1. 三文魚切碎，加入青瓜粒、西芹粒、番荽碎、洋葱粒、番茄粒、醃料拌勻，醃15分鐘。

2. 沙律菜加少許橄欖油、黑醋、黑椒碎、鹽拌勻。

3. 將裝飾用的青瓜片放在碟，放上小圓模，加入芒果肉、三文魚混合物、沙律菜後脫模。

4. 在碟上放上小番茄、檸檬，伴上薄多士，淋少許黑醋即可。

貼士：

三文魚不要切得太碎，不然就沒有口感了。

4.日式三文魚扒

燒三文魚扒一點都不難，而精髓就在於和風味的汁料，再加上木魚花，有如到餐廳吃的日式料理，好高貴！

材料

三文魚扒100克、京葱2條、紅蘿蔔（切絲）/白蘿蔔（切絲）各20克、小番茄1粒、乾紫菜碎/木魚花/橄欖油各少許

醃料：日本豉油/味噌/清酒/生粉各1茶匙、薑汁1/2茶匙、青芥末1/3茶匙、水2茶匙、鹽/黑椒碎各少許、味醂1湯匙

汁料：日本豉油/味醂各1湯匙、清酒1茶匙、高湯4湯匙、鹽/糖各1/8茶匙、黑椒碎少許

做法

1. 京葱切段；小番茄切去底部。

2. 三文魚扒加入醃料，醃15分鐘備用。

3. 開中火，燒熱鑊落少許橄欖油，煎香三文魚扒後盛起，放入預熱至200℃焗爐，焗8分鐘。

4. 原鑊開中火，將京葱煎香盛起，再將小番茄輕輕煎熟。

5. 碟上放京葱、小番茄、紅蘿蔔絲、白蘿蔔絲、三文魚扒。

6. 開中火，將汁料材料煮至微滾後熄火，淋在三文魚扒上，灑上紫菜碎、木魚花裝飾即可。

貼士：

想三文魚更加外脆內嫩，可將魚略煎後放入焗爐。

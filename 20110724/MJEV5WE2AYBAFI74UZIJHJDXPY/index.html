煮題　芝士甜品
有料到
青芥末　專攻鼻竇

即日式芥末，由山葵的根部磨茸而成。其辛辣味會刺激鼻竇，卻不刺激舌頭。

無鹽牛油　味道易控制

古時，牛油加鹽以保存其新鮮度，自從雪櫃出現後，牛油有鹽與否同樣易於保存。無鹽的奶脂味濃，宜用來烹飪或做甜品。

忌廉芝士　做蛋糕沙律

屬軟芝士，味道溫和，約含33%乳脂，一般用來做芝士蛋糕或配沙律同吃。

意大利芝士　香滑濃郁

約含65%乳脂。由生產至到期日通常不超過半年，在香港的大型超市較易買到。

巴馬臣芝士　帶果香

屬硬芝士，帶刺鼻的異味，入口卻帶濃烈的乾果香味！

淡忌廉　注意溫度

打拂時要注意溫度，於室溫下待太久，淡忌廉可能會打拂不起或不挺身。其保存期限較短，應盡早使用。

芥末芝士凍餅

芥末用來做甜品咁激？原來用芥末來做芝士蛋糕，鹹鹹哋的芝士加辣辣哋的芥末，味道其實好夾。想像唔到？自己整嚟試試啦！

材料

海綿蛋糕（15厘米）1片、綠色鏡面啫喱適量

芝士餅底：原味芝士餅底碎75克、無鹽牛油（放軟）35克

芥末芝士：魚膠粉7克、蛋黃1隻、鮮奶70毫升、忌廉芝士（放軟）180克、砂糖30克、芥末醬22克、淡忌廉（已打拂）200克、冧酒15毫升

做法

1. 先將餅底碎、無鹽牛油混合，倒入已包好錫紙的餅模內，壓平備用。

2. 將鮮奶、魚膠粉、蛋黃混合，以熱水坐溶，拌成魚膠啫喱。

3. 忌廉芝士、砂糖混合後，以電動打蛋器打至鬆軟，加入淡忌廉、芥末醬、冧酒、魚膠啫喱拌勻成芥末芝士。

4. 將一半的芥末芝士倒入餅模中，放上海綿蛋糕，再倒入餘下芥末芝士，放入雪櫃雪1小時。

5. 在蛋糕面抹上綠色鏡面啫喱，加上裝飾即可。

焗藍莓芝士條

芝士食得多會滯，配上生果就可中和膩滯感。焗過的芝士藍莓條，蛋味濃香、芝士味更突出，再加上藍莓的清新，好正！

材料

原味芝士餅底碎90克、無鹽牛油（放軟）45克

藍莓芝士：忌廉芝士（放軟）200克、砂糖25克、芝士香油2滴、低筋麪粉/粟粉各5克、雞蛋1隻、淡忌廉（已打拂）50克、檸檬汁1茶匙、無鹽牛油（已溶）12克、藍莓果餡100克

裝飾：新鮮藍莓125克、藍莓果餡40克

做法

1. 餅底碎、無鹽牛油混合，倒入已包好錫紙的餅模內，壓平成餅底備用。

2. 忌廉芝士、砂糖、芝士香油混合，以電動打蛋器打至鬆軟，加入低筋麪粉、粟粉、淡忌廉、雞蛋打勻，加入檸檬汁、無鹽牛油拌勻成藍莓芝士。

3. 將餅模放在焗盤中，倒入一半的藍莓芝士，鋪上藍莓果餡，倒入餘下藍莓芝士抹平，在焗盤中注入5毫米清水。

4. 將芝士餅放入預熱至175℃焗爐，焗45分鐘，出爐後待凉。

5. 新鮮藍莓、藍莓果餡混合，鋪在芝士條上作裝飾，再切成8件即可。

貼士

藍莓果餡內含70%藍莓，與塗麪包的藍莓果醬有所不同，果餡可於烘焙店買到。

意大利芝士木糠布甸

講起木糠布甸唔一定要去澳門食，其實自己整都得，Charles話木糠同布甸要平均分配，咁先滑溜。

材料

魚膠粉3克、蛋黃1隻、鮮奶40毫升、意大利芝士/淡忌廉（已打拂）各160克、芝士香油2滴、煉奶30毫升、杏仁酒15毫升、檸檬汁少許、原味芝士餅底碎85克

做法

1. 鮮奶、魚膠粉、蛋黃混合，坐熱水拌成魚膠啫喱，備用。

2. 意大利芝士、淡忌廉、煉奶，以電動打蛋器打至滑身，加芝士香油、檸檬汁、杏仁酒、魚膠啫喱拌勻成意大利芝士慕絲。

3. 凍杯中加入少許原味芝士餅底碎，倒入芝士慕絲，重複以上步驟三次，然後放雪櫃雪30分鐘，最後加上裝飾即可。

雞肉芝士撻

這個芝士撻，芝士夾着雞肉、蘑菇，最緊要是個批底，啖啖都咁香濃，好有質感。

材料

餅底：無鹽牛油40克、砂糖/雞蛋（拂勻）各15克、鹽11/2茶匙、牛油香油2滴、淡奶3毫升、低筋麪粉25克、高筋麪粉35克

雞肉餡：無鹽牛油10克、乾葱頭（切碎）1/2粒、熟雞髀肉（拆絲）50克、蘑菇片/火腿（切粒）各20克、海鹽少許

芝士餡：忌廉芝士（放軟）160克、砂糖10克、低筋麪粉12克、淡忌廉（已打拂）70克、車打芝士（刨碎）/巴馬臣芝士碎各20克

其他：巴馬臣芝士碎（裝飾）25克、高筋麪粉適量

做法

1. 先做餅底，將無鹽牛油、砂糖、鹽、香油以打蛋器打勻，分兩次加入蛋液、淡奶，打至鬆軟呈乳白色，加入高筋麪粉、低筋麪粉拌勻。

2. 將混合物以手搓成滑身麪糰，灑少許高筋麪粉，壓成直徑約18厘米圓塊，鋪在餅模上。

3. 準備雞肉餡，開中火，燒熱鑊下無鹽牛油，先爆香乾葱頭，加雞髀肉、蘑菇片、火腿粒炒香，灑上海鹽炒勻備用。

4. 做芝士餡，將忌廉芝士、砂糖、低筋麪粉混合，以打蛋器打勻成鬆軟的麪糊，加忌廉、車打芝士碎和雞肉餡拌勻成餡料。

5. 將餡料倒入餅模中，灑上巴馬臣芝士碎，放入預熱至175℃焗爐，焗45分鐘，出焗後灑上裝飾芝士碎即可。

煮題　暖心團圓迎元宵

有料到
糯米粉　黏膩難消化

用糯米製成，黏性較高，通常用來整糕點和湯圓，不過其黏膩而較難消化，腸胃不適者小心。

薄荷　能驅趕蚊蟲

可作烹煮、藥食或製成藥膏。對初起感冒、頭痛、目赤有治療功效。在家栽種薄荷，更可驅趕蚊蟲。

西班牙香腸　風乾毋須烹煮

英文是Chorizo，用豬脂肪與較粗糙的豬肉做成，以紅辣椒粉和鹽調味。新鮮的要煮熟才可進食，已風乾的則毋須烹調。

煙肉　高鹽脂肪多

一般煙肉會用豬脊肉來做，而美國煙肉則用豬腹的肉，帶有較多脂肪。煙肉含鹽量較多，不宜多吃。

煉奶　防骨質疏鬆

含蛋白質、鈣質，有助預防骨質疏鬆，其水份大約是鮮牛奶的四分之一，常被用於製作甜品。

迷迭香　增強記憶力

地中海菜式常用香料，可增強記憶力，其葉內所含的精油，可增加食物的風味。

1.脆炸甜草莓

西方情人節通常會用士多啤梨來做甜品，不妨將這個士多啤梨炸脆，換成同樣甜蜜蜜的中式甜品吧！

材料

草莓20粒、麪粉375克、鹽/乾酵母各1/4茶匙、糖霜1湯匙、啤酒/冰水各半杯、橄欖油1茶匙、煉奶25毫升、蛋黃/蛋白各2隻

做法

1. 將250克麪粉、糖霜一同過篩，加入乾酵母、鹽、冰水、啤酒、橄欖油拌至糊狀，再用電動打蛋器打成滑身麪粉漿，放室溫發酵約1小時。

2. 蛋白用電動打蛋器打至企身，備用。

3. 在麪粉漿中加入蛋黃拌勻，再加入已打起的蛋白，拌勻成麪糊。

4. 草莓去蒂，用竹籤挖空中心，將煉奶注入草莓內，然後將草莓沾上餘下麪粉，再均勻地沾上麪糊。

5. 開中火，燒熱油至200℃，每次放入數粒草莓，炸至金黃色，撈起瀝油。最後灑上少許糖霜即可。

貼士

士多啤梨沾麪粉時，宜在唧煉奶的那面，沾多點麪粉，以免炸時煉奶流出。

2.迷迭香燒雞髀雞翼

這道迷迭香燒雞髀雞翼，取其「比翼雙飛」的意頭，吃過後，每日定會跟情人過得甜甜蜜蜜。

材料

迷迭香4條、雞髀肉1塊、雞全翼4隻、煙肉8片、黃燈籠椒1隻、黑椒少許

醃料：鹽/糖各1茶匙、橄欖油2湯匙、豉油1湯匙

做法

1. 將雞全翼拆骨；雞髀肉切大件，然後一同用醃料拌勻，醃15分鐘，備用。

2. 燈籠椒去蒂、去籽、切角，備用。

3. 將煙肉包着雞件。用迷迭香串起雞翼、燈籠椒、煙肉雞件，備用。

4. 開中火，燒熱平底鑊，放入雞串，每面煎5分鐘，盛起，灑上黑椒即可。

貼士

因為煙肉包着雞肉煎香，煎時會有油份滲出來，煎時可毋須加油。

3.心形梨焗煙肉香腸

形狀似心形的紅安琪梨，最適合做元宵應節食品，加上西班牙香腸和芝士，似足歐洲的街頭小吃，特別又浪漫非常。

材料

紅安琪梨2個、芝士/煙肉（切碎）各2片、西班牙香腸（切粒）50克、檸檬（榨汁）1/4個

做法

1. 先將紅安琪梨左邊切去大片，右邊切去小片；將大片切成心形，備用。

2. 於紅安琪梨左邊的中心刺幾刀，再用湯匙挖空果肉，紅安琪梨備用。果肉切粒，備用。

3. 燒熱平底鑊，下煙肉、西班牙香腸炒至半乾身，再放果肉快炒，加入檸檬汁，盛起。

4. 將炒熟的材料放入挖空的紅安琪梨中，梨面鋪上芝士，與梨蓋一同放入預熱至200℃焗爐，焗15分鐘即可。

4.薄荷朱古力湯圓

即搓即煮的湯圓，口感非常煙韌，一口咬落去，朱古力馬上流出，特別心甜。

材料

糯米粉50克、生粉適量、熱水30毫升、薄荷朱古力碎適量、冰糖3湯匙、薄荷葉1條、清水400毫升

做法

1. 糯米粉過篩，加熱水搓成粉糰。

2. 枱上灑少許生粉，放上粉糰，搓成長條形，然後分成10小份，逐一包入朱古力碎，搓成湯圓。

3. 煲滾半鍋水，放入湯圓，煮至浮起，撈起盛於碗中備用。

4. 煲滾清水，放入冰糖煮至溶後熄火，放薄荷葉焗3分鐘，將薄荷糖水放入碗裏即可。

小王子偷入廚搵食　甜到漏婚Cake

英國威廉王子與未婚妻凱蒂將於4月29日共諧連理，皇室世紀婚禮上，將有威廉王子兒時最愛的朱古力餅乾蛋糕作結婚蛋糕，別小看這個簡單的蛋糕，它包含了王子小時候的快樂回憶。而這個蛋糕的原創人前英國皇室行政主廚CarolynRobb，近日便訪港並大方公開了這個蛋糕的秘方！

記者：謝翠玲
攝影：楊錦文
示範：前英國皇室行政主廚CarolynRobb

Profile

CarolynRobb出生於南非，以卓越成績在英國著名的廚藝學校CordonBleuCookery畢業，於1989至2000年任職皇室御廚。過去曾為查理斯王子、已故戴安娜王妃及威廉王子主理宮內外膳食。現時，Carolyn是世界知名有機食品廚師，她在離宮後曾在各地多間米芝蓮食府任職。

甜品做獎賞

曾擔任英國皇室行政主廚長達11年的CarolynRobb，對於皇室成員的口味可算瞭如指掌，基本上他們的口味及飲食習慣也偏向健康，在後花園種有新鮮蔬果來食用，而像威廉王子最愛的朱古力餅乾蛋糕這種濃甜的甜品，算是SpecialTreats（特別獎賞），要在特別日子才可享用，但當然，畢竟小王子也是人一個，在王子還是小小男孩時，他們也會像其他男孩一樣，偷偷跑進廚房，有禮地央求Carolyn給他們朱古力蛋糕等零食，而這時Carolyn則會聰明地把這難題交回給他們的母親戴安娜王妃！
為了照顧皇室成員的胃口，Carolyn的任務便是經常隨查理斯王子出訪預備食物。她最難忘的便是1997年香港回歸時，她隨着查理斯王子訪港，為他準備了一頓難忘的晚餐。而在皇宮多年，Carolyn非常清楚各皇室成員的口味及忌諱，例如不可在料理中放蒜頭，怕吃完會有難聞的口氣，又或是端上桌的食物不可太熱等，不多花點精神也不行。

親嚐皇室菜

沙田新城市廣場特別邀請了皇室御廚CarolynRobb來港炮製非凡品味菜式，顧客由即日起於商場以電子貨幣消費滿$15,000，即可登記於母親節期間（5月7或8日）在新城市廣場1期7樓THEMENU@Level7空中花園內品嚐由CarolynRobb親自主理、價值$3,000（供兩位享用）的「皇室菜」。

金箔皇室朱古力蛋糕

今次王子大婚中備受注目的結婚蛋糕「金箔皇室朱古力蛋糕」，原身就是Carolyn為小威廉王子創製的朱古力餅乾蛋糕。這本是Carolyn媽媽的家傳食譜，到她手上改良成獨有配方，王子小時候每逢生日或特別日子，都指定要吃這個蛋糕。

材料

牛油266克、金黃糖膠150克、熱朱古力粉/可可粉各33克、消化餅碎333克、無花果乾（切粒）/紅莓乾（切粒）/杏脯乾（切粒）/提子乾/生核桃（切粒）/杏仁片/開心果（切半）各37克

裝飾：金箔/朱古力醬各適量、開心果仁碎1茶匙

做法

1. 牛油、糖膠、熱朱古力粉和可可粉加入小煲中，以小火加熱至牛油溶，其間要不停攪拌至均勻。

2. 加入提子乾、杏脯乾、紅莓乾、無花果乾繼續攪拌，熄火。

3. 再加入消化餅碎、杏仁片、生核桃和開心果，拌勻後倒入圓模中，壓平，放雪櫃雪30分鐘至硬身。

4. 取出蛋糕，鋪上朱古力醬，再加上開心果碎和金箔作裝飾即成。

貼士

喜歡吃甜味重些，可加入較多熱朱古力粉，並減少可可粉的比例。

烤乳龍蝦伴特級魚子醬

威廉王子另一心頭好便是這個烤乳龍蝦伴特級魚子醬，用上牛油煎煮龍蝦，配上魚子醬，切合王子身份。

材料

波士頓龍蝦（烚熟去殼）500克、新鮮麪條63克、牛油45克、蘆笋（烚熟）6條、迷你大蒜2條、鹽少許、紫蘇葉1棵、魚子醬1茶匙、西柚皮/血橙皮/檸檬皮/青檸皮絲各少許　

調味料：黑胡椒/鹽各適量

醬汁：蔬菜清湯225毫升、西柚/血橙/檸檬/青檸各半個

做法

1. 先製作醬汁，將西柚、血橙、檸檬和青檸榨汁。將蔬菜清湯以大火煮至剩下一半份量，加入榨好的果汁，其間輕壓一下讓果肉爆開。

2. 15克牛油加入鑊中，下龍蝦肉，以小火煎香，下黑胡椒和鹽調味，再加入蘆笋和迷你大蒜同煮。加入半份醬汁至鑊中，讓龍蝦和蔬菜入味，備用。

3. 另備一煲，下10克牛油至熱水中，加少許鹽，下鮮麪條煮至軟身即撈起。

4. 另備一煲，加入餘下的牛油和醬汁，部份西柚皮絲、血橙皮絲、檸檬皮絲及青檸皮絲，下麪條和紫蘇葉同拌炒。

5. 將麪上碟，加上魚子醬，放上龍蝦，再以少許紫蘇葉和果皮裝飾即成。

特級巴馬火腿沙律伴25年極品黑醋

這道清簡的沙律是哈里王子的至愛，做法簡單，用料卻精細，選用了25年極品黑醋伴食，即時提升了味道的層次。

材料

巴馬火腿3片、新鮮無花果1個、烘松子1茶匙、迷你Mozzarella芝士3粒、火箭菜13克

沙律醬汁：核桃油11/2湯匙、極品黑醋1茶匙、海鹽/黑胡椒少許

做法

1. 先將巴馬火腿排碟。無花果去蒂，切塊，鋪在碟上。

2. 大碗中放入火箭菜，加入核桃油、黑胡椒、海鹽和半茶匙黑醋拌勻，放在碟上。

3. 在菜面放上迷你Mozzarella芝士和松子，最後加入半茶匙黑醋即成。

紅噹噹講意頭　缽磨魚茸好糰圓

雖然日本人慣於慶祝新曆年，但作為一年之始，他們在整個一月也講求意頭，例如會煮通窿食物如蓮藕、獅子狗魚蛋等，寓意前途無阻事事順景；炸吉列豬扒就有勝利之意；而雞肉就代表豐收。今日Gohsan教大家做幾道日式意味着「團圓」的意頭菜，讓大家新一年合家團圓！

今日煮腦

張一豪（日本小吃達人）

曾任職飲食記者逾10年，操流利日語及深愛日本美食，曾在港開設章魚燒店、糖果店等。

煮題　日式新年糰糰圓圓
有料到
蒟蒻　幼鹽洗鹼味

熱量相當低，含水溶性纖維，能幫助腸胃蠕動。烹調蒟蒻前宜用幼鹽擦淨，可去除其鹼水味。

牛蒡　長生蔬果

日本人認為有「長生」之意，因它含十幾種氨基酸，可增強體力，其豐富的膳食纖維，有助整腸排毒。揀牛蒡宜揀重手，無爛無根無爆口為佳。

日本里芋　解凍即煮

其實是去了外皮的細芋頭，於大型日式超市的急凍櫃可買到，解凍便能煮食，方便快捷。

豬柳肉　脂肪少夠柔嫩

是脊骨下一條與大排骨相連的瘦肉，因其運動量少、脂肪含量少，是豬肉中最柔嫩的部位，煎炸炒焗都十分適宜。

荷蘭豆　煮太久變黃

有益脾健胃、生津止渴，烹調時約需2至3分鐘，可保持顏色鮮嫩，太耐會變爛和變黃。

白玉粉　與糕粉相同

以白鑊炒熟了的糯米粉，日本是用糯米去殼加水研磨成漿，脫水乾燥後而成，與一般使用的糕粉相同。

汁煮赤魚鯛糰子

日本鯛魚的日文叫「medetai」，跟喜慶「omedetai」讀音相近，故鯛魚對日本人來講是好意頭。

材料

赤魚鯛魚身（400克）2件、牛蒡4支、剝皮沙鯭魚肉200克、荷蘭豆少許

調味料：鹽1茶匙、清酒/味醂各1湯匙

煮汁：薑3片、清酒/味醂/醬油/糖各2湯匙、水150毫升

做法

1. 剝皮沙鯭魚肉切碎，放入磨缽磨茸，下鹽、味醂、清酒拌勻，並分為四份。

2. 將赤魚鯛魚身切片，放上微波爐紙，再放魚肉茸，然後揑成球狀。

3. 燒滾水，放入魚球，蒸8分鐘，取出攤凉備用。

4. 牛蒡刨皮，切成5吋左右長，浸水備用。

5. 燒熱鑊，下煮汁料，加入牛蒡，煮滾後，放入魚球，不停淋上汁料煮至上色，最後下荷蘭豆煮至軟身。

6. 將魚球上碟，放上牛蒡、荷蘭豆即成。

貼士

打魚肉用磨缽效果最好，可打出膠質，若無也可用攪拌機，或用刀拍爛魚肉後用刀背剁碎也可。

吉列飯糰

日文的吉列豬扒「Katsu」，同勝利的「Katsu」同音，今次Gohsan做一口吉列，用飯糰包住食，真係勝利又團圓呀！

材料

豬柳肉100克、日本飯4碗、壽司紫菜1張

配料：鹽/胡椒粉各少許、蛋1隻、麪粉2湯匙、日本麪包糠4湯匙

做法

1. 將豬柳肉切成一口大小，撒上鹽、胡椒粉醃味。

2. 蛋漿拂勻，加入麪粉成粉漿。豬肉沾粉漿，再上麪包糠。

3. 燒滾油至180℃，將豬柳肉炸約3分鐘至全熟。

4. 沾濕雙手，取出適量的日本飯，包入一件吉列豬肉握實，包成飯糰。

5. 最後包上一片紫菜即可上碟。

貼士

想飯糰更有味，可在搓飯糰時，先用少許幼鹽捽勻雙手再包，包出來的飯就更有味。

筑前煮

筑前煮是日本過年意頭菜之一，將很多食物放在一齊煮，而日本人一定會放雞肉，因雞肉的日文「Tori」有收穫豐富之意。

材料

雞扒（切件）200克、蓮藕（切片）100克、里芋/竹輪獅子狗魚蛋各8粒、牛蒡100克、甘笋（切角）80克、荷蘭豆8隻、乾冬菇8隻、蒟蒻磚（250克）1件、幼鹽適量、熱水300毫升

汁料：木魚出汁素21/2茶匙、清酒4湯匙、味醂1湯匙、糖3湯匙、醬油5湯匙

做法

1. 冬菇加入300毫升熱水，浸軟身，浸水留用。將冬菇去椗，切成細件備用。

2. 蒟蒻磚用幼鹽略捽，以水冲洗，切成一口大小，放入滾水汆水，撈起。

3. 牛蒡刨皮，切斜片，浸水備用。

4. 燒熱鑊下油，放入雞肉炒至五成熟後，下冬菇、甘笋、牛蒡、蓮藕、里芋、蒟蒻不停炒勻，再下清酒、味醂、木魚出汁素、糖、冬菇浸水、醬油。

5. 滾起後，下獅子狗魚蛋，收細火煮15-20分鐘，最後再下荷蘭豆略煮1分鐘即可。

白玉紅豆湯

日本每年除夕都有「紅白」歌唱比賽，因為紅白二色是日本人的新年吉祥顏色，所以連糖水都喜歡以紅白二色作配搭。

材料

紅豆茸500克、水1公升、鹽少許

白玉：白玉粉180克、水160毫升

做法

1. 先做白玉，將白玉粉、水攪勻後，倒上枱搓成粉糰，平均分成小粒。

2. 煮滾水，加入白玉粒約煮兩分鐘至熟透，撈起過冰水。

3. 將水倒入煲，下紅豆茸拌勻，煮滾，放少許鹽調味。

4. 將白玉放入碗，倒入紅豆湯即成。

貼士

嫌做白玉麻煩，可用燒過的日本方年糕mochi都得。

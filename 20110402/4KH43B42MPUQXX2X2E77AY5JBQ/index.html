入廚字典：慢熱三文魚　煎皮脆腩半生熟

香港人愛吃三文魚，一年自外地入口就有過千噸。平日常吃的是挪威三文魚，但來自蘇格蘭、澳洲和愛爾蘭有機三文魚，就肯定少見。原來各地三文魚的味道、口感都各有特色，不論哪種食法，已經令人流晒口水！

記者：吳韻菁
攝影：陳陶鈞
示範：DickWong（香港銅鑼灣皇冠假日酒店Kudos行政總廚）

Profile

曾於英國及意大利任職客席總廚、學習廚藝及餐廳營運的DickWong，曾為水警總部酒店、AquaRestaurant&Bar等餐廳服務，並參與餐廳廚房設計。

挪威種最肥美

很多人都說挪威三文魚只宜煙熏，到底是真是假？Dick解答，「任何水域出產的三文魚都可以做煙熏或刺身，亦可烹飪其他菜式。其烹調和食法主要以部位劃分，因不同部位的口感味道明顯不同。」
一般食用部位分頭、尾、背、腩和鰭。頭和鰭常用於日式烹調法，如煮湯、鹽燒；西式就以腩、尾及背為主，魚腩肥美可做刺身或半生熟煮法；魚尾較細和少脂肪，適合切碎來做他他菜式；魚背肉較多和瘦，可較完整地烹煮。
原來不同水域的三文魚亦有分別，「以腩部來說，挪威和蘇格蘭的較肥美，前者油感重，後者肥美得來爽口。」而挪威三文魚產量最高，價格較便宜；蘇格蘭是較貴價的品種；澳洲的飼養水域較廣，活動較多；愛爾蘭的則以有機飼料飼養，這三地的品種價錢較貴，香港餐廳較少購入。

三文魚

頭：油香嫩滑
鰭：油香肥美
背：肉多較瘦
腩：肥美油多
尾：脂肪少

A.慢煮三文魚

澳洲三文魚腩以慢火香煎至半熟，食落不覺得油膩，再伴以馬鈴薯黑魚子汁料，又香濃又Creamy。

材料

有皮三文魚腩200克、西洋菜適量

馬鈴薯汁：馬鈴薯（去皮切片）1個、鮮忌廉50毫升、黑魚子10克、牛油/海鹽/黑胡椒碎各少許

紅菜頭啫喱：紅菜頭3個、水2公升、橙（去皮起肉）2個、肉桂條1支、大菜粉5克、粗鹽/黑胡椒碎各少許

做法

1. 紅菜頭、水、橙肉、肉桂條一同煮滾至剩1公升份量，加入大菜粉、鹽、黑胡椒拌勻熄火，待凉至凝固，備用。

2. 將三文魚有皮的一邊，以小火煎至魚皮和肉略熟，盛起備用。

3. 馬鈴薯烚熟後瀝水壓成茸，混合鮮忌廉、牛油、海鹽、黑胡椒碎拌勻，以小火煮至稠身熄火，加入黑魚子拌勻。

4. 將黑魚子馬鈴薯汁料淋於碟上，再放上三文魚、紅菜頭啫喱及西洋菜即可。

貼士

只煎有魚皮的一面，讓熱力滲至魚肉，才可做到半生半熟的效果。

B.軟三文魚配香橙及番茄沙沙

愛爾蘭三文魚以低溫烹調法來烹煮，以斷開其纖維，放入口中即似棉花般散開，軟綿中加上香橙和番茄的清新味道，盡是好吃。

材料

三文魚背肉（切粒）200克、細葱適量

醃料：葡萄籽油100毫升、橄欖油80毫升、白胡椒1/2茶匙、羅勒葉5片、百里香1棵、蒜頭（剁茸）1個、西芹（切段）1棵、甘笋（切片）1條

香橙番茄沙沙：橙肉（切粒）1個、番茄（剁茸）2個、細葱（切碎）10克、初榨橄欖油10毫升、紅酒醋5毫升

做法

1. 三文魚加醃料拌勻，放雪櫃雪兩小時。

2. 醃好後將三文魚置室溫下解凍，將細葱撒在三文魚的其中一面。

3. 放入預熱至60℃焗爐，焗20分鐘，每5分鐘在三文魚上掃上少許橄欖油。

4. 把香橙番茄沙沙材料混合，備用。

5. 碟上放上焗好的三文魚，放上香橙番茄沙沙，加上裝飾即可。

C.三文魚他他

選用挪威三文魚腩，生吃口感嫩滑，Dick特地加入了紅菜頭啫喱，令口感層次更豐富。

材料

三文魚腩（切細粒）100克、熟蟹肉/番茄茸各30克、細葱50克、檸檬（刨皮）1個、初榨橄欖油10毫升、紅酒醋/黑胡椒碎/海鹽各少許

牛油果慕絲：牛油果（去核切粒）2個、牛奶300毫升、檸檬（榨汁）1個、魚膠片（凍水浸軟）1塊

紅菜頭啫喱：紅菜頭3個、水2公升、橙（去皮起肉）2個、肉桂條1支、大菜粉5克、粗鹽/黑胡椒碎各少許

做法

1. 先做紅菜頭啫喱，將紅菜頭、水、橙肉、肉桂條煮滾至剩1公升份量，加入大菜粉、鹽、黑胡椒碎拌勻熄火，待凉至凝固，備用。

2.將慕絲的材料用攪拌機打勻，放入唧槍內備用。

3. 番茄茸、初榨橄欖油、黑胡椒碎、海鹽、紅酒醋混合成番茄醬；碟上放圓模，放入番茄醬、熟蟹肉。

4. 將三文魚粒、黑胡椒碎、海鹽、檸檬皮、細葱混合，放於蟹肉上，取走圓模。

5. 在面唧上牛油果慕絲，放入少許紅菜頭啫喱和細葱作裝飾即可。

貼士

唧上了牛油果慕絲後要盡快食用，否則慕絲會塌下。

D.三文魚尼哥斯沙律

一般尼哥斯沙律都是用吞拿魚，Dick今次用蘇格蘭三文魚來代替，用香片茶葉熏過後，味道非常好！

材料

三文魚背肉（切件）/香片茶葉/黃糖/白米各100克、青瓜絲/青瓜乳酪/黑芝麻各少許、鵪鶉蛋5隻

尼哥斯沙律：紅洋葱30克、迷你馬鈴薯/四季豆各100克、車厘茄50克、黑橄欖10克、初榨橄欖油15毫升、葡萄醋10毫升、海鹽少許

做法

1. 鵪鶉蛋灼熟後去殼切半；紅洋葱切絲；迷你馬鈴薯切角灼熟；四季豆切段灼熟，備用。

2. 將茶葉、黃糖、白米混合，倒入鋁造小盤中，再放入煲內。開中火加熱，冚上蓋至有煙出，熄火。

3. 煲內放一鐵架，放上三文魚，蓋上蓋焗8分鐘至三文魚呈淺啡色，備用。

4. 將尼哥斯沙律材料混合，先置於碟上，放上鵪鶉蛋，再放上三文魚、青瓜絲、青瓜乳酪和黑芝麻即可。
